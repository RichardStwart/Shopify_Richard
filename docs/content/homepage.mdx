## Why Use Medusa

Medusa aims to provide developers with the best development experience on their journey to build a unique ecommerce experience. Developers and businesses have full control over their tech stack and the logic behind essential ecommerce features.

Whether you're looking for a platform that provides you with the essential infrastructure to build on top of, or a platform that allows you to customize every aspect of your ecommerce store to build a bespoke commerce experience, Medusa is the ideal choice.

## Medusa Components

Medusa is composed of three main components: the Medusa server, the Medusa admin, and a storefront.

### Medusa Server

The Medusa server contains all the core logic and data processing of your ecommerce platform. Its APIs are built with abstraction to allow developers to extend them. The Medusa server can also be integrated to third-party services and tools through plugins.

### Medusa Admin

The Medusa Admin is an intuitive admin dashboard that allows businesses to manage their ecommerce store and its data including order management, product management, store settings, and much more.

You can alternatively build your own admin dashboard using the Admin REST APIs.

### Ecommerce Storefront

The storefront is where your customer view your products and place their orders. Medusa provides Next.js and Gatsby starter storefronts that you can use as building blocks for your storefront. You can also build your storefront from scratch using the Store REST APIs.
