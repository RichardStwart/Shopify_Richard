# Medusa Documentation

Medusa is a set of ecommerce tools and modules that can be used to build unique ecommerce experiences.

Medusa provides the essential building blocks that developers can put together to create a powerful ecommerce store. Developers have full control over their tech stack and the logic behind the ecommerce features.
