# Core Concepts

The Medusa server is the core that powers the ecommerce platform. It handles the storage and processing of data in your store. It’s built on top of Node.js and Express, and it integrates with a database and an events system.

:::info

If you’re looking for Medusa’s ecommerce features, check out the [Features and Guides](../features-and-guides/overview.mdx) section.

:::
